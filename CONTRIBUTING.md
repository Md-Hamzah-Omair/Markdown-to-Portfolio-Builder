# Contributing to Markdown-to-Portfolio Builder

First off, thanks for taking the time to contribute! 

## How to add new Themes
1. Create a new React component in `src/themes/`.
2. Ensure it accepts `data` and `htmlContent` as props.
3. Register your theme in `src/components/ThemeWrapper.jsx`.
4. Submit a Pull Request!

## Development Process
- Fork the repo.
- Create a feature branch.
- Run `npm install`.
- Test your changes with `npm run dev`.
- Submit a PR with a clear description.
